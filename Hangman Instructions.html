Hangman Instructions



To Play Press The Start Button Below



  1. Enter a letter from the alphabet into the Guess a Letter text box
  2. Press the guess button
  3. If the guessed letter is in the randomly selected phrase it substitutes the – for the letter, if it is in the phrase multiple times, all of the letters will appear
  4. If the guessed letter isn’t in the phrase it will add one to wrong guesses, and tells the user they have guess wrong by changing the hangman image, and adding the wrong letter to a list of wrong letters displayed on the screen
  5. When all letters in the phrase have been guessed correctly, the user has finished the game
  6. If the man is completely hanged on the image, a message will appear telling the user they have been hanged and you have lost the game so will not be able to continue
